🛡️ 验证码配置 (YesCaptcha)

使用浏览器扩展自动解决 FunCaptcha 验证码

🌐 代理配置

支持 http、https、socks5，启用后浏览器注册流程将通过代理访问。

🛡️ 账号操作代理

仅用于账号凭证导入后的检查与刷新、获取成员、移除成员、邀请成员和产品分配。不会用于注册浏览器、邮箱 API 或 Token 池。

🔌 自动入池配置

注册成功后，Cookie 会自动导入已启用的 Token 池；某个池失败时会自动尝试下一个池。

📈 子账号池监测

开启后定时查询选中 Token 池的已耗尽账号；命中内部子账号池后，先移除旧子号，再为所属母号自动补 1 个新子号。
